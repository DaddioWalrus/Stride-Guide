The glance that matters

Mid-walk you look up for under a second. Today the turn instruction is a thin translucent bar above a dense stats panel — the priority is inverted. The redesign promotes the maneuver to the hero and demotes ETA/distance to a slim bar. Shown in dark mode, where a nav app spends its evenings.

Identity · three directions

Pick a point of view

All three ship as the same token set — switching direction is a variable swap, not a rebuild — and each ships light + dark. The mockups above use Horizon for the UI and Nocturne as its dark theme.

Trailheadnature-forward

Spruce green, limestone neutrals, a waymark-orange CTA. OS-map / hiking calm.

ProMost distinctive; unmistakably an outdoors app.
ConGreen can compete with parks on the map tiles.
Recommended

Horizonevolved · energetic

A confident indigo with a warm coral for go-moments. Familiar lineage, sporty energy.

ProLow-risk evolution; blue stays distinct from map greys; superb dark mode.
ConClosest to today — the wins ride on type + hierarchy.

Nocturnedark-first

Near-black canvas, a luminous lime/cyan that pops on UI and map alike.

ProBest for night + glare + battery; most premium.
ConLight variant needs careful tuning; boldest risk.

Roadmap · sequenced by risk

How it ships

main deploys straight to production with no staging, so each tier is independently revertable and lands in order. Tier 1 is safe enough to ship today.

Tier 1~same day · CSS

Quick wins

  1. Contrast fix to AA H
  2. 44px touch targets H
  3. aria-labels on icon buttons H
  4. Microcopy pass — modes, verbs, placeholders
  5. Type-scale tokens — foundation for the rest
  6. Focus-visible rings
Tier 2~days · structural

The upgrades

  1. Colour tokens + dark mode H
  2. Line-icon set replaces emoji H
  3. Nav hierarchy flip — maneuver hero H
  4. Explicit unit toggle H
  5. Retire pixel-positioning for a docked sheet
Tier 3the reveal

Bold identity

  1. Adopt Horizon + Nocturne dark H
  2. Motion system — phase transitions, press physics M
  3. Elevation tiers — a real depth language
Four decisions to move forward: the identity direction (Horizon, or see the others rendered first); dark-mode default vs. manual toggle; bespoke vs. open-licensed icons; and whether to land Tier 1 on the live app now while we spec the rest.
